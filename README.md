## Installation

1. Clone this repository
2. Run `composer install`
3. Import database or run Drupal installation
4. Enable theme and module in admin panel

## Custom Code

### Theme: `uni_theme`
Located in `web/themes/custom/uni_theme/`
- Custom responsive CSS
- Twig page template

### Module: `uni_events`
Located in `web/modules/custom/uni_events/`
- Custom Block Plugin displaying upcoming events
